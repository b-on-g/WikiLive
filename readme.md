# WikiLive

Модуль живой вики-системы — текст и таблицы как единый инструмент для совместной работы и управления знаниями.

## Запуск (Docker)

```bash
docker-compose up --build
```

Приложение будет доступно по адресу: http://localhost:9082/bog/WikiLive/app/-/test.html

## Запуск (локально)

```bash
# Из корня MAM-монорепы
npx mam bog/WikiLive
npm start
```

Приложение будет доступно по адресу: http://localhost:9080/bog/WikiLive/app/-/test.html
